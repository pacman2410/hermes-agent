# Career Lab HQ Future Platform Roadmap

This folder preserves future platform systems. These should be designed now but not built until the Validation MVP proves willingness to pay, repeated engagement, and score-driven outcomes.

## Tier 2 - Design Now, Build Later

- CRM
- Analytics layer
- Coach portal
- Admin portal
- Authentication
- Stripe integration
- Expanded AI knowledge operations
- User memory operations

## Tier 3 - Preserve For Later

- Community
- Courses
- Certifications
- Enterprise platform
- Corporate coaching
- Group coaching marketplace
- Leadership academies
- Corporate subscriptions

## Build Gate

Do not build Tier 2 or Tier 3 until:

- Paid diagnostics convert.
- Weekly check-ins are used.
- Users request continued tracking.
- Human review demand is proven.
- A clear ICP emerges.
